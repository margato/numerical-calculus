Método dos trapézios¶

Autor: Osvaldo César Cuba Margato¶

Linguagem utilizada: Python 3.6¶

Introdução¶

Pelo teorema fundamental do cálculo, seja f uma função contínua no intervalo [a, b], temos que:

\begin{equation}\int_{a}^{b} f(x) dx = F(b) - F(A)\end{equation}

em que, F(x) é uma função primitiva de f(x). A partir disso, podemos afirmar que a integral definida dessa função representa a área A abaixo da curva e pode ser denotada como:

\begin{equation}A = \int_{a}^{b} f(x) dx\end{equation}

Visualização geométrica:

Saber a área de uma curva é muito útil no mundo real, como definir o trabalho W de uma força a partir do deslocamento em x metros. Ao descobrir a área da curva temos o trabalho, ou seja, devemos integrar a força em relação ao deslocamento. \begin{equation}W = \int_{a}^{b} f(x) dx\end{equation}

Motivação¶

Algumas funções não possuem uma função primitiva, logo, pelos teoremas e postulados, não é possível resolver.

Exemplo¶

A função erro (ou função erro de Gauss) é uma função não-elementar muito usada em estatística. \begin{equation} erf(x) = \dfrac{2}{\sqrt\pi} \int_{0}^{t} e^{-x^2} dx = \dfrac{1}{\sqrt\pi} \int_{-t}^{t} e^{-x^2} dx \end{equation}

A função: \begin{equation} \int_{-t}^{t} e^{-x^2} dx \end{equation} Não possui uma função primitiva, então para esse caso, podemos aplicar o método dos trapézios para calcular a área embaixo da curva.

Definição¶

Dado um intervalo [a, b] e n iterações, podemos aproximar a área A da curva pela área de trapézios. Função geral:

\begin{equation} \text{Área dos trapézios } = A_i = \dfrac{h}{2}[f(x_0) + 2(f(x_1) + ... + f(x_{n-1})) + f(x_n)] \\ \text{Sendo } h = \dfrac{b-a}{n} \end{equation}

Para calcular a área exata, subtraimos o erro do método: \begin{equation} \text{Área exata } = A = A_i - \dfrac{h^3}{12} f''(\xi) \\ \text{Sendo }\xi \text{ um número entre a e b} \end{equation}

Aplicação:¶

Nesse trabalho, o método foi aplicado na função abaixo, no intervalo [-1, 1]: \begin{equation} \int_{a}^{b} e^{-x^2} dx \end{equation}

%matplotlib inline
import numpy as np
import pandas as pd
import math
import matplotlib.pyplot as plt

def f(x):
    # f(x) = e^(-x²)
    return math.exp(-x**2)

a = float(input("A: "))
b = float(input("B: "))
n = int(input("Iterações: "))
n = n + 1
h = float((b - a) / n)

ys = []
xs = np.linspace(a, b, n, endpoint=True).tolist()
print(f"Intervalo: [{a}, {b}], iterações: {n - 1}, passo: {h}")
for xi in xs:
    ys.append(f(xi))
points = {
    'x': xs,
    'y': ys
}
df = pd.DataFrame(points)
df.index = df.index + 1

Função que retorna a área da curva pelo método dos trapézios¶

In [4]:
def trapezes(h, ys):
    y_sum = sum(ys[1:(n - 1)])
    return (h/2) * (ys[0] + ys[-1] + 2 * y_sum)
