Método dos trapézios¶

Autor: Osvaldo César Cuba Margato¶

Linguagem utilizada: Python 3.6¶

Introdução¶

Pelo teorema fundamental do clculo, seja f uma funo contnua no intervalo [a, b], temos que:

\begin{equation}\int_{a}^{b} f(x) dx = F(b) - F(A)\end{equation}

em que, F(x) uma funo primitiva de f(x). A partir disso, podemos afirmar que a integral definida dessa funo representa a rea A abaixo da curva e pode ser denotada como:

\begin{equation}A = \int_{a}^{b} f(x) dx\end{equation}

Visualizao geomtrica:

Saber a rea de uma curva muito til no mundo real, como definir o trabalho W de uma fora a partir do deslocamento em x metros. Ao descobrir a rea da curva temos o trabalho, ou seja, devemos integrar a fora em relao ao deslocamento. \begin{equation}W = \int_{a}^{b} f(x) dx\end{equation}

Motivação¶

Algumas funes no possuem uma funo primitiva, logo, pelos teoremas e postulados, no possvel resolver.

Exemplo¶

A funo erro (ou funo erro de Gauss) uma funo no-elementar muito usada em estatstica. \begin{equation} erf(x) = \dfrac{2}{\sqrt\pi} \int_{0}^{t} e^{-x^2} dx = \dfrac{1}{\sqrt\pi} \int_{-t}^{t} e^{-x^2} dx \end{equation}

A funo: \begin{equation} \int_{-t}^{t} e^{-x^2} dx \end{equation} No possui uma funo primitiva, ento para esse caso, podemos aplicar o mtodo dos trapzios para calcular a rea embaixo da curva.

Definição¶

Dado um intervalo [a, b] e n iteraes, podemos aproximar a rea A da curva pela rea de trapzios. Funo geral:

\begin{equation} \text{rea dos trapzios } = A_i = \dfrac{h}{2}[f(x_0) + 2(f(x_1) + ... + f(x_{n-1})) + f(x_n)] \\ \text{Sendo } h = \dfrac{b-a}{n} \end{equation}

Para calcular a rea exata, subtraimos o erro do mtodo: \begin{equation} \text{rea exata } = A = A_i - \dfrac{h^3}{12} f''(\xi) \\ \text{Sendo }\xi \text{ um nmero entre a e b} \end{equation}

Aplicação:¶

Nesse trabalho, o mtodo foi aplicado na funo abaixo, no intervalo [-1, 1]: \begin{equation} \int_{a}^{b} e^{-x^2} dx \end{equation}

%matplotlib inline
import numpy as np
import pandas as pd
import math
import matplotlib.pyplot as plt

def f(x):
    # f(x) = e^(-x)
    return math.exp(-x**2)

a = float(input("A: "))
b = float(input("B: "))
n = int(input("Iteraes: "))
n = n + 1
h = float((b - a) / n)

ys = []
xs = np.linspace(a, b, n, endpoint=True).tolist()
print(f"Intervalo: [{a}, {b}], iteraes: {n - 1}, passo: {h}")
for xi in xs:
    ys.append(f(xi))
points = {
    'x': xs,
    'y': ys
}
df = pd.DataFrame(points)
df.index = df.index + 1

Função que retorna a área da curva pelo método dos trapézios¶

In [4]:
def trapezes(h, ys):
    y_sum = sum(ys[1:(n - 1)])
    return (h/2) * (ys[0] + ys[-1] + 2 * y_sum)
